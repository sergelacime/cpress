Compressez vos fichiers
sans les envoyer en ligne

cPress est une application desktop open source qui réduit la taille de vos images, documents, PDF, audio et vidéo — entièrement en local, sur votre machine.

Open source · MIT Tauri 2 + Rust macOS · Windows · Linux 100 % local
↓ Télécharger Voir sur GitHub

Fonctionnalités

Une interface simple et efficace pour compresser vos fichiers au quotidien, sans compromettre votre vie privée.

🔒

Compression 100 % locale

Aucune donnée n'est envoyée sur Internet. Tout le traitement se fait sur votre ordinateur, hors ligne.

📂

Glisser-déposer

Déposez vos fichiers ou sélectionnez-les en lot. Traitez plusieurs fichiers d'un coup avec suivi de progression.

🎚️

Curseur de qualité

Ajustez le compromis taille / fidélité de 1 à 100 selon vos besoins — web, archivage ou impression.

👁️

Aperçu intégré

Prévisualisez images, vidéos, audio, PDF et documents avant et après compression, côte à côte.

📋

Historique persistant

Retrouvez vos dernières compressions avec les économies réalisées, enregistrées localement en JSON.

🌓

Thème clair & sombre

Interface adaptée à votre environnement de travail, avec bascule instantanée entre les deux modes.

Formats pris en charge

Détection automatique du type de fichier et moteur de compression adapté à chaque catégorie.

Catégorie Formats Moteur
Images JPEG, PNG, WebP image, oxipng (Rust)
Documents Office DOCX, XLSX, PPTX, ODT, ODS, ODP Recompression ZIP + images internes
Archives ZIP Optimisation des entrées
PDF PDF qpdf (optionnel) ou lopdf
Vidéo MP4, MOV, MKV… FFmpeg (sidecar embarqué)
Audio MP3, WAV… FFmpeg (sidecar embarqué)

Les fichiers compressés sont créés à côté de l'original avec le suffixe _compressed (ex. : photo.jpg → photo_compressed.jpg).

Comment ça marche

Quatre étapes simples pour réduire la taille de vos fichiers sans quitter votre bureau.

1

Ajoutez vos fichiers

Glissez-déposez ou choisissez un ou plusieurs fichiers depuis l'explorateur.

2

Réglez la qualité

Utilisez le curseur pour définir le niveau de compression souhaité.

3

Lancez la compression

Suivez la progression en temps réel. Chaque type de fichier est traité par le moteur adapté.

4

Récupérez le résultat

Ouvrez le fichier compressé dans le Finder ou comparez-le dans l'aperçu intégré.

Vos fichiers ne quittent jamais votre machine

Contrairement aux services en ligne, cPress ne télécharge rien vers un serveur. Idéal pour documents sensibles, archives professionnelles et usage hors connexion.

Installation

Téléchargez la dernière release pour votre système, ou compilez depuis les sources.

Depuis les sources (développeurs)

git clone https://github.com/sergelacime/cpress.git
cd cpress
pnpm install
pnpm tauri dev

Prérequis : Node.js 18+, pnpm, Rust stable. Pour un installeur de production : pnpm tauri build (FFmpeg est téléchargé automatiquement).
